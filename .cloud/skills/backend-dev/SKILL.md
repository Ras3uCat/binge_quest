# Backend Development Skill

## Stack
- Supabase (Postgres + Edge Functions)
- Firebase (fallback)
- REST APIs
- Stripe integration
- JWT-based auth

## Rules
- Schema changes via migrations
- RLS enabled
- Backend is source of truth
- Tests required
- Services handle API calls
- Repositories handle data/business logic
- Always write tests

## When to Use
Whenever modifying backend APIs, auth, database, or business logic.
